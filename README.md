# üåê Project: Recommender System

## Introduction
**Goal**: Build a recommender system capable of handling cold start and dynamic user/movie addition scenarios.

## Main tasks
1. **Cold start**: user selects a genre ‚Äî we return top-5 by popularity.
2. **Few ratings**: several viewed films ‚Äî we search for similar ones (genres + tags + relevance).
3. **Few ratings**: <5 ratings, or >5, but <3 positive.
4. **Full recommendations**: >5 ratings ‚Äî we cluster the user vector.
5. **Promotion/blocking**: a filtering system at the search results level.
